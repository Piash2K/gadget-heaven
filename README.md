## ğŸ—‚ï¸ Data Handling and Management

The application manages data using the following approaches:

* **Context API**
  Used as the primary data management solution to provide global access to shared data.

* **CartProduct Context**
  Stores and manages products added to the shopping cart, enabling users to add, remove, and sort cart items globally.

* **WishProduct Context**
  Manages wishlist products, making wishlist data easily accessible and manageable throughout the application.

## âœ¨ Key Features of Gadget Heaven

* **Dynamic Product Display and Details View**
  Displays a collection of gadgets with individual product detail pages showing specifications and relevant information.

* **Cart and Wishlist Management**
  Users can add products to the cart or wishlist, view selected items, and remove them when needed.

* **Purchase Modal with Toast Notifications**
  A purchase confirmation modal is displayed upon checkout, while toast notifications provide instant feedback for user actions.

* **Star Rating Display**
  Product ratings are visually represented using the ReactStars component, helping users make informed decisions.

* **Price Sorting in Cart**
  Allows users to sort cart items based on price (highest to lowest) for better purchase management.
